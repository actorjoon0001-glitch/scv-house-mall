🏘️ 마을 모델 관리

카탈로그 원본은 그대로 두고, 마을·랜딩 표시용 설정만 저장합니다. 저장하면 손님 화면은 새로고침 시 반영됩니다.

노출사진표시 이름가격(만원)평수 소속 존큐레이터특징 설명대표 사진 URL

· 빈 칸 = 카탈로그 원본 값 사용 · 가격은 만원 단위(예: 6840) · 큐레이터 "자동"은 존 내 순서대로 여/남 교차

📋 모델 취합

전체 모델의 최종 표시 정보(관리자 수정 반영)를 한 표로 취합해 봅니다. 보고·공유용 CSV로 내려받을 수 있어요.

#사진모델명존배치 칸평수가격 큐레이터노출비고

· 이 표는 손님에게 실제 보이는 값 기준(이름/가격/존 수정 반영) · 숨김 모델은 회색으로 표시

🗺️ 구역 배치 관리 (위에서 본 지도)

집을 클릭해 선택한 뒤 원하는 칸을 클릭하면 이동합니다(칸이 차 있으면 서로 교체). 드래그로도 옮길 수 있어요. 화살표는 집의 정면 방향(↓=남쪽 통로)입니다.

선택된 집 없음
🏢 부스 관리: 빈 칸을 클릭해 선택한 뒤 상태를 지정하세요. 부스 칸에는 모델이 자동 배치되지 않아요.

· 초록 테두리 = 관리자가 칸을 고정한 집 · 위가 북쪽, 각 존 아래쪽 줄이 통로(입구) 쪽 · 저장은 왼쪽 💾 전체 저장 버튼

🤝 입점 업체 관리

B2B 입점 업체의 회사 정보·담당자·계약 상태를 관리합니다. 부스를 배정하면 3D 마을 부스 간판에 회사명이 자동 반영돼요.

부스 칸은 🗺️ 배치·부스 지도의 "부스 관리 모드"에서 먼저 만들 수 있어요
회사명담당자연락처이메일업종/존 상태계약 시작계약 종료배정 부스메모

아직 등록된 업체가 없습니다. "➕ 업체 추가"로 시작하세요.

· 상태를 계약/입점완료로 두고 부스를 배정하면 마을 부스에 회사명이 표시됩니다 · 저장은 왼쪽 💾 전체 저장 버튼

🏗️ 계약 고객 시공 현황

계약 고객별로 시공 단계·현장 사진·일정·안내를 올립니다. 올리면 손님의 내 집 현황(/my) 화면에 바로 반영돼요. 현장에서 폰으로 접속해 사진을 올려도 됩니다.

고객연락처상태단계사진문의

등록된 시공 고객이 없습니다. "➕ 고객 등록"으로 첫 계약 고객을 추가하세요.

· 사진은 자동으로 축소(최대 1280px)돼 올라가요 · 손님은 본인 번호+닉네임으로만 자기 현황을 볼 수 있어요

🎨 존 관리

존 표시 이름·색상과 파트너 존의 부스 이름을 바꿉니다. 3D 마을의 간판·바닥·게이트 색에 그대로 반영돼요. (빈 칸 = 기본값)

· 존 구분(어떤 모델이 어느 존인지)은 모델 관리 표의 "소속 존"에서 관리 · 저장은 왼쪽 💾 전체 저장 버튼

🎪 체험존 포털

마을 남서쪽 체험존에 줄지어 배치되는 체험 포털 목록입니다. 이름을 바꾸거나 숨길 수 있고, 새 체험 공간이 생기면 포털을 추가하면 돼요.

· URL이 없는 포털은 "준비 중"으로 표시됩니다 (예: VR룸) · 저장은 왼쪽 💾 전체 저장 버튼
